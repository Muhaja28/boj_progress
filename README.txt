===========================================
BOJ Progress Checker – 사용 설명서 (README)
===========================================

이 프로그램은 BOJ(백준 Online Judge) 학생 계정을 기반으로
- 특정 문제집의 풀이 진도 확인
- 푼 문제 목록 / 안 푼 문제 목록
- 문제별 BOJ 하이퍼링크 제공
을 지원하는 로컬 FastAPI 웹 애플리케이션입니다.

-------------------------------------------
1. 요구 환경
-------------------------------------------
이 프로그램을 실행하기 위해서는 다음이 필요합니다.

- Python 3.10 이상
- pip (보통 Python 설치 시 포함됨)
- 인터넷 연결
- 웹 브라우저 (Chrome, Edge 등)

Windows에서 Python 명령은 보통 `py` 로 실행합니다.


-------------------------------------------
2. 설치 방법 (ZIP 다운로드 기준)
-------------------------------------------

(1) GitHub 저장소에서 ZIP 다운로드
    → "Code" 버튼 클릭 → "Download ZIP"

(2) ZIP 파일 압축 해제
    예: boj_progress 폴더가 생성됨

(3) 압축 해제한 폴더로 이동
    예: C:\Users\USER\Desktop\boj_progress

(4) 필요한 라이브러리 설치
    터미널(PowerShell 또는 CMD)에서 아래 실행:

        py -m pip install -r requirements.txt

    설치 시간은 약 10~30초 소요됩니다.


-------------------------------------------
3. 프로그램 실행
-------------------------------------------

아래 명령을 터미널에서 실행하세요:

    py -m uvicorn main:app --reload

성공하면 터미널에 다음과 비슷한 메시지가 나타납니다:

    Uvicorn running on http://127.0.0.1:8000

이제 웹 브라우저에서 아래 주소로 접속합니다:

    http://127.0.0.1:8000/

웹 페이지가 열리면 프로그램 사용을 시작할 수 있습니다.

※ 서버 종료는 터미널에서 CTRL + C


-------------------------------------------
4. 프로그램 사용 방법
-------------------------------------------

(1) BOJ 학생 ID를 입력
(2) 문제집을 선택
(3) “조회” 버튼 클릭

조회 결과:
- 학생의 문제집 풀이 진도
- 푼 문제 목록 + BOJ 이동 링크
- 안 푼 문제 목록 + BOJ 이동 링크


-------------------------------------------
5. workbooks.json 수정 방법 (문제집 변경)
-------------------------------------------

문제집 목록은 `workbooks.json` 에 저장되어 있습니다.

예시:
{
  "graph_basic": {
    "name": "그래프 기초 문제집",
    "problems": [1260, 2606, 2667, 2178]
  },
  "dp_basic": {
    "name": "DP 기초 문제집",
    "problems": [1003, 1463, 9095]
  }
}

- 문제 번호 추가/삭제는 JSON을 직접 수정하면 됩니다.
- EXE가 아니라 Python 실행 버전에서는 수정 즉시 반영됨.


-------------------------------------------
6. 오류가 발생할 경우
-------------------------------------------

1) `py` 명령이 없다 → Python이 설치되지 않은 상태  
   https://python.org 에서 Python 3 다운로드 후 설치  
   설치 시 *Add Python to PATH* 체크 필요

2) 패키지 설치 오류 → 관리자 권한 터미널 실행

3) 브라우저에서 페이지가 안 뜨면  
   - 서버 실행 여부 확인 (터미널 메시지)
   - 127.0.0.1 주소 오타 확인


-------------------------------------------
7. 종료
-------------------------------------------

프로그램 종료는 터미널에서:

    CTRL + C
